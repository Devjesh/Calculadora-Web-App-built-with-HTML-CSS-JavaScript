Responsive Web Calculator

Uma calculadora web responsiva construída com HTML, CSS e JavaScript. A calculadora suporta operações básicas como adição, subtração, multiplicação, divisão e porcentagem.

Funcionalidades

Operações básicas: Adição (+), Subtração (-), Multiplicação (*), Divisão (/)
Cálculo de porcentagem (%)
Limpeza de tela (AC)
Avaliação de expressões com a tecla "="


Tecnologias Utilizadas

HTML - Estruturação da interface do usuário.
CSS - Estilização e design responsivo.
JavaScript - Lógica e funcionalidades da calculadora.
